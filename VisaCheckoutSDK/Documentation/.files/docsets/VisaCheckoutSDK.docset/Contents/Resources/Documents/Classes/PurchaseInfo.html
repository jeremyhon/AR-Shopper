PurchaseInfo

public class PurchaseInfo: NSObject

The PurchaseInfo class is used for you to send the Visa Checkout SDK the detailed information regarding your customers’ purchase. Use this class if you need to pass more detailed information other than the amount and currency.

  • referenceCallId

    This value should be set if a successful CheckoutResult was received AND you need to modify the purchase by invoking the Visa Checkout SDK again for the same purchase.

    For instance, if the user successfully completes the Visa Checkout process, CheckoutResult.callId will be populated with a string that represents this purchase. If, then, you need to make adjustments to the exact same purchase, you must be sure to set this value to the one obtained from CheckoutResult.callId before the user attempts to checkout again.

    An example of setting the referenceCallId with the value from a CheckoutResult instance

    let checkoutResult: CheckoutResult = ...
let purchaseInfo = PurchaseInfo(...)

purchaseInfo.referenceCallId = checkoutResult.callId
