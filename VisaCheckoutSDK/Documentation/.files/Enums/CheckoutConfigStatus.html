public enum CheckoutConfigStatus: Int

The result code indicating the status of a completed configure() call of the VisaCheckoutSDK class.

  • debugModeNotSupported

    You have attempted to run the Visa Checkout SDK in debug mode while simultaneously configuring using Environment.production as the ‘environment` value.

  • invalidAPIKey

    You have supplied an incorrect API Key value for the apiKey property. Be sure to obtain a valid API Key from your developer account and also remember to use the correct API Key for the correct environment.

    For example, you will have at least 2 API Keys. One for the Environment.sandbox type and one for the Environment.production type. Be sure you are using the correct one and also be sure you have copied the value correctly.
